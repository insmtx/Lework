# 标书最终检查清单

> 生成投标文件后必须逐项核对。废标项为红色警报级别。

## 🔴 废标项检查（红色警报）

- [ ] 投标函中项目名称与招标文件完全一致
- [ ] 投标函中项目编号与招标文件完全一致
- [ ] 投标有效期 ≥ 招标文件要求
- [ ] 投标保证金金额满足要求
- [ ] 投标保证金缴纳形式满足要求（基本账户转账）
- [ ] 资格审查资料齐全
- [ ] 营业执照在有效期内
- [ ] 资质证书在有效期内
- [ ] 业绩证明满足数量/金额/时间要求
- [ ] 合同关键页齐全（双方盖章页+合同签订时间+关键信息页）
- [ ] 发票金额与合同匹配
- [ ] 法人授权书有法人签字并加盖公章
- [ ] 授权委托书期限覆盖投标有效期
- [ ] 所有星号条款已响应
- [ ] 报价未超过最高限价
- [ ] 分项报价齐全无遗漏
- [ ] 报价大小写一致

## 🟡 格式检查

- [ ] 封面信息正确（项目名称、编号、投标人、日期、正/副本标识）
- [ ] 目录页码与实际内容一致
- [ ] 最后一页页码 = 总页数
- [ ] 无空白页
- [ ] 无其他项目名称残留
- [ ] 无其他项目编号残留
- [ ] 无其他日期残留
- [ ] 签字盖章位置标注清晰
- [ ] 正本每页有公章（如要求）
- [ ] 骑缝章位置正确
- [ ] 正文标题编号连续且无跳跃

## 🟢 内容一致性检查

- [ ] 分项名称在方案、偏差表、报价表中一致
- [ ] 偏差表引用的章节位置真实存在
- [ ] 报价合计 = 投标函总价
- [ ] 方案中没有其他项目/公司名称
- [ ] 人员姓名/证书/社保一致且在职
- [ ] 公司/产品资质日期在有效期内

## 🔵 其他检查

- [ ] 页码：插入分页符后验证目录页码未断裂
- [ ] 索引表：评标专家可快速定位评分项
- [ ] 点对点应答：详见页码指向正确
- [ ] 电子版文件可正常打开
- [ ] 光盘/U盘文件完整
